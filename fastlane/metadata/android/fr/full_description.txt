<i>The Translator</i> est une application simple pour aider à traduire d'autres applis.  <b>Paramètres</b>  - Une interface simple mais élégante pour afficher chaque ligne pour une langue. - Les utilisateurs sont intelligemment guidés pour éviter les erreurs. - Partage direct (après avoir enregistré) des chaînes avec les dévleoppeurs. - Appuie long pour supprimer une ligne particulière. - Thèmes sombres et clairs. - Traduit en plusieurs langues (aidez-moi à traduire cette appli — le strings.xml original est disponible sur <a href='https://github.com/sunilpaulmathew/Translator/blob/master/app/src/main/res/values/strings.xml'>GitHub</a>). - Et bien plus  <b>Comment ça marche ?</b>  - Récupérez le « strings.xml » original (ou l'URL de ce fichier) via votre développeur (ou depuis le dépôt source si l'appli est open-source). - Importez celui-ci (depuis « sdcard » ou l'URL) dans The Translator (Paramètres --> Importer un fichier « string »). - L'appli affichera chaque ligne à traduire dans des entrées séparées. - Cliquez sur chacune et remplacez le texte d'origine par la traduction dans votre langue. - Suivez les instructions à l'écran pour éviter les erreurs (la couleur rouge indique une erreur). - Quand vous avez fini, enregistrez votre travail en cliquant sur le bouton de sauvegarde en bas (ou dans Paramètres --> Voir le fichier « string »). - Envoyez le nouveau ficher xml généré dans « sdcard » à votre développeur et attendez patiemment la prochaine mise à jour.
